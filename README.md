
# HTTP(S) - Client 

## Program can:
* Send post request and get answer
* Send get request and get answer
* Send information from file by post request
* Save answer in file
* You can add: cookie, user-agent, headers and reference 
  
### How it works:
 In Terminal: python make_request.py url + args

### Args use:
-d + "string"

-f + "file name"

-e + "reference"

-O + "file name"

-A + "User-Agent"

-c + "cookie"
